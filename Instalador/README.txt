¡Gracias por desargar Win Bool: Procesador de operaciones booleanas!

Si ud. descargó el software de otra fuente, le recomiendo descargarlo del sitio oficial:
https://github.com/GiacomoMadrid/win-bool

Si ud. tiene Java 20 instalado puede usar el .jar sin problema.
Si ud. eliminó el acceso directo del escritorio puede copiar el acceso directo "Win Bool.exe"

Si ud. borra el acceso directo puede crear uno de "WinBool.bat".
El archivo .bat ejecuta el JRE que viene con el software y a partir de ahí ejecuta el archivo .jar, de este modo ud. no tiene la necesidad de descargar Java, sin embargo si ud. posee JDK 17 - JDK 20 debería bastarle con el .jar.
